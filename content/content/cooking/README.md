# 🫕 요리

### **요리 방법**

<div align="left"><figure><img src="../../../.gitbook/assets/cook.png" alt=""><figcaption><p>'/요리' UI</p></figcaption></figure></div>

1. '/레시피'를 통해 요리 레시피를 확인한다
2. 원하는 레시피를 요리 상점에서 구매한다
3. 요리 재료를 준비한 후 '/요리 재료' 또는 '/요리 가마솥'에 재료를 넣어준다
4. '/요리'에서 만들 요리를 클릭한 후 제작하기 버튼을 누른다
5. 채팅에 원하는 요리의 숫자를 입력한다
6. 요리가 끝난 후 '/요리 가방'에서 요리를 꺼내면 끝\~



### **요리 장비**

요리 상점 NPC하루토를 우클릭하면 요리사의 비법을 얻을 수 있습니다\
기본 슬롯은 2칸이며, 잠금 해제 열쇠로 슬롯을 한 줄 확장할 수 있습니다

<div align="left"><figure><img src="../../../.gitbook/assets/제목 없음-9 (1).png" alt=""><figcaption><p>요리사의 비법</p></figcaption></figure></div>

<div align="left"><figure><img src="../../../.gitbook/assets/잠금해제열쇠 (1).png" alt=""><figcaption><p>잠금 해제 열쇠</p></figcaption></figure> <figure><img src="../../../.gitbook/assets/고급해제열쇠 (1).png" alt=""><figcaption><p>고급 잠금 해제 열쇠</p></figcaption></figure></div>

일반 잠금 해제 열쇠는 30% 확률로 슬롯을 확장할 수 있고, \
고급 잠금 해제 열쇠는 100% 확률로 슬롯을 확장할 수 있습니다.
